# BrainRoc

A BF interpreter written in Roc.

```shell
$ cat hello-world.bf
++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.
$ brainroc hello-world.bf
Hello World!
```

## Usage

To download and run some example BF programs, install the [Roc compiler](https://roc-lang.org) and [`just`](https://github.com/casey/just), then run:

```shell
just examples
```
